What is Clover?

Clover is a Substrate-based Polkadot parachain. Clover is committed to providing easy-to-use blockchain infrastructure and creating a one-stop comprehensive infrastructure platform, which reduce the threshold and cost for developers, and bring great returns to developers.

Our Vision
  • Bring Most Benefit to Developers
  • From dApp to eApp (Eternal App)
  • Connecting Blockchain to Everything
Clover OS / Frame

The bottom layer of Clover OS is built on blockchain. Application states are built on Polkadot network, and the storage is built on IPFS.

Smart Contract
Smart Contract protocol layer supports one-stop cross-chain deployment and calls of dApp.
  • Polkadot
  • Ethereum
  • Tron
  • Eos
Clover Front End (UI/UX)
Front End protocol layer supports front-end multi-OS adaptation and operation.
  • ios
  • Android
  • Linux
  • Windos LoT
Clover Back End
Back End protocol layer supports various basic Defi protocols.
